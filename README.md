## 🔑 Features
- **Google Login** (OAuth 2.0)  
  - Login using your NEU Google account (`jcesperanza@neu.edu.ph`).
- **Role‑Based Access Control**  
  - **Regular User**:  
    - After validation, greeted with:  
      > “Welcome to NEU Library!”
  - **Admin**:  
    - View visitor statistics by **day, week, or custom date range**.  
    - Statistics displayed in **cards**.  
    - Filter by:  
      - Reason for visiting  
      - College  
      - Employee status (teacher/staff)

---

## 🛠️ Tech Stack
- **Frontend**: React + Material UI  
- **Backend**: Node.js + Express  
- **Database**: MongoDB / PostgreSQL  
- **Authentication**: Firebase Authentication (Google Sign‑In)  
- **Deployment**: Vercel (frontend) + Render/Heroku (backend)
